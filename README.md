# HBLHacks
Scripts for auto login to Ivy.
## For Python
### Prerequisites
*Python 3* installed on *MacOS* / *Windows*.
### Instructions for Setup
Using ```requirements.txt```:
```
python -m pip install --upgrade pip
pip install -r requirements.txt
```
Alternatively:
```
python -m pip install --upgrade pip
pip install pyautogui
pip install time
pip install sys
```
### Instructions to Run
```
python main.py
```
